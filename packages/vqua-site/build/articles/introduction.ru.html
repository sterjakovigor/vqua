Вступление

Vqua это JavaScript бибилиотека для создание веб-интерфейсов.
Ориентированная на создания изморфных/унивесральных приложений.

Экосистема

Серверный пререндер

  • vqua-router - универсальный роутер
  • vqua-navigation - браузерная навигация
  • vqua-server - серверный пререндер

Другое

  • dom2vqua - преобразует dom ноды в vqua ноды
  • html2vqua - преобразует html строку в vqua ноды
  • vqua2string - преобразует vqua ноды в строку
  • vqua-interpolate - преобразует переменные в текстовых нодах
  • vqua-utils - утилиты использумые в vqua
